API
===

Translate
---------

The link to be used for the API call is this one: `anstranslation.ddns.net:5000/translate_with_parameters?text=cat&metric=True&boolMult=False <anstranslation.ddns.net:5000/translate_with_parameters?text=cat&metric=True&boolMult=False>`_

Parameters:

* text: text to be translated
* boolMult: return multiple translations
* metric: return unsupervised metrics

Returns json with:

* Translation output: returns multiple translations if requested,
* Score: the translation likelihood of the model,
* Score2: sentence bert similarity score with english label.
